🚀 NOMI CYBER-X NETWORK TOOLKIT

A professional Termux-based network and system utility toolkit designed for Android users. This toolkit provides quick access to useful device and network information through a simple menu-driven interface.

---

✨ Features

- 📱 System Information
- 🌐 Network Information
- 📶 Wi-Fi Information
- 🌍 Public IP Lookup
- 🔎 DNS Lookup
- 📡 Ping Test
- 📝 Report Export
- 📂 Log Viewer
- 🎨 Colorful Terminal Interface
- ⚡ Lightweight & Fast
- 📱 Optimized for Termux

---

📥 Installation

Update packages:

pkg update -y && pkg upgrade -y

Install Git:

pkg install git -y

Clone Repository:

git clone https://github.com/nomi-cracker/NOMI-CYBER-X-NETWORK-TOOLKIT.git

Open Project Directory:

cd NOMI-CYBER-X-NETWORK-TOOLKIT

Give Execute Permission:

chmod +x install.sh

Install Requirements:

bash install.sh

Run Toolkit:

bash toolkit.sh

---

🖥️ Usage

After launching the toolkit, select an option from the menu:

[1] System Information
[2] Network Information
[3] WiFi Information
[4] Public IP Lookup
[5] DNS Lookup
[6] Ping Test
[7] Save Report
[8] View Logs
[9] About
[0] Exit
